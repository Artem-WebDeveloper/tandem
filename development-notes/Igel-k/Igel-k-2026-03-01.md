# Регистрация новых пользователей

## Задачи:
- Разработать интерфейс модального окна для регистрации новых пользователей.
- Реализовать логику взаимодействия с API для регистрации пользователей.
- Настроить визуальное оформление страницы с учётом переключения тем.
- Добавить на страницу авторизации пользователей переход к регистрации новых пользователей.

## Ход работы:

#### Разработка модального окна регистрации
Вместо создания отдельной страницы для регистрации, я решила использовать модальное окно. Я добавила компонент с модальным окном, использую комипонент ```Dialog``` из библиотеки ```Material UI```.

#### Интеграция с API
Для создания пользователя была написана функция, выполняющая POST-запрос через библиотеку ```axios```.

#### Связка компонентов
Я добавила на страницу авторизации пользователей кнопку, которая открывает диалоговое окно. После успешной регистрации, форме авторизации передаётся ```username``` и далее пользователю нужно только ввести корректный пароль.

Итог: Приложение поддерживает полный цикл: создание нового пользователя, авторизация пользователя и выход из учётной записи.
