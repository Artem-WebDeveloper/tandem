В компоненте `Header` - `const { user, logout } = useAuthStore()` подписывает компонент на весь store. Тогда `Header` перерисуется при любом изменении в `auth.store`, даже если `user/logout` не менялись. Более безопасным вариантом будет - `const user = useAuthStore((state) => state.user); const logout = useAuthStore((state) => state.logout);`. Это то, что я понял изучая Zustand.
