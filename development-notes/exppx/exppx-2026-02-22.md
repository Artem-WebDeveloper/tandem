## Ход работы

Начал с переноса палитры, которую сделал вчера на макет сайта в Figma. Смотрелось не плохо, только изменил пару цветов.  
Теперь нужно перенести все в тему в Material UI. Поискал, как это лучше сделать и получилось все без особых проблем. Единственный момент был с тем, как повесить стиль с градиентным фоном на body. Для этого нужно было добавить переписанное правило для body из CssBaseline, в которое вставить стиль фона из палитры:

```typescript
return createTheme({
  palette,
  typography,
  components: {
    MuiCssBaseline: {
      styleOverrides: (theme) => ({
        body: {
          backgroundImage: theme.palette.backgroundGradient,
          backgroundAttachment: "fixed",
          backgroundRepeat: "no-repeat",
          backgroundSize: "cover",
        },
      }),
    },
  },
});
```
