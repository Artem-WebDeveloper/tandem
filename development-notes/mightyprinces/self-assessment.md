# Описание работы

В рамках проекта я занималася разработкой Code Ordering квиза и страницы Results квизов

## Основные задачи и вклад

Два личных Feature Component

### 1. Разработка Code Ordering Widget

Я спроектировала и реализовала компонент квиза с перетаскиванием строк кода в нужном порядке, чтобы получить корректную функцию. Для этого использовала библиотеку dnd-kit.

### 2. Страница Results квизов

Сделала немного отличающийся UI для адекватного вывода результатов для каждого типа квизов - Code Ordering, Code Completion, Async Sorter, Single Choice, True False.
